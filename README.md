# auto-code-refactor
基于 AI 大模型的自动化代码重构工具，自动扫描项目问题、生成重构方案、自动编写测试用例。

## 项目介绍
这是一个面向开发者的自动化代码重构工具，主要用于：
- 自动扫描老项目中的代码问题、不规范写法
- 自动生成安全、可直接使用的重构代码
- 自动生成单元测试，保证代码质量
- 支持批量处理整个项目，大幅提升维护效率

## 核心功能
- 自动代码审查与问题检测
- 一键代码重构与优化
- 自动生成测试用例
- 支持大型项目批量处理

## 后续计划
本项目计划基于 **MiMo-V2.5-Pro** 大模型进行开发，利用其百万级上下文能力，实现完整项目级别的代码分析与重构。
