# auto-code-refactor
基于大模型的自动化代码重构工具，解决遗留项目维护成本高、重构风险大的痛点。

## 🚀 项目简介
这是一个面向开发者的AI驱动代码重构工具，核心目标是：
- 自动扫描存量项目中的技术债、不规范写法和潜在bug
- 基于项目架构规范生成安全、可直接运行的重构方案
- 自动编写单元测试，完成重构后的闭环验证
- 支持批量处理，大幅提升大型项目的维护效率

## ✨ 核心功能
- **代码问题检测**：自动识别代码中的重复逻辑、不规范命名、潜在性能问题
- **智能重构生成**：根据上下文自动生成优化后的代码片段
- **测试用例生成**：为重构后的代码自动编写单元测试，确保行为一致
- **批量处理能力**：支持整个项目文件的批量扫描与重构建议生成

## 📋 使用场景
- 老项目技术债清理
- 代码规范统一与风格对齐
- 大型项目重构前的风险评估与方案生成
- 代码评审的自动化辅助工具

## 🛠️ 后续开发计划
本项目后续将基于 **MiMo-V2.5-Pro** 大模型进行迭代优化：
- 利用其百万级上下文能力，实现完整项目级别的代码分析
- 提升复杂业务逻辑的重构理解能力
- 优化测试用例生成的覆盖率与准确性
- 开源完整的工具代码与使用文档，供开发者社区使用
