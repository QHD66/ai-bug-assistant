# ai-bug-assistant
基于大语言模型的智能调试助手，自动解析报错日志、定位代码问题并生成修复方案。

## 项目简介
这个工具可以帮助开发者快速定位代码bug，你只需要粘贴报错日志和相关代码，它就能：
- 自动分析报错原因，定位问题代码位置
- 生成可直接运行的修复代码
- 编写对应的单元测试，验证修复效果

## 核心功能
- 🔍 错误日志解析：支持处理超长报错栈信息
- 🛠️ 自动修复生成：根据上下文生成修复代码
- ✅ 单元测试生成：自动生成验证用例
- 📝 调试报告：输出清晰的问题分析与修复说明
